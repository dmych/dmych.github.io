# Google Tasks

Простое решение, доступное пользователям Google «из коробки». Существенный плюс: интегрирован с [[ĝardeno/Google Calendar\|гугловским календарем]], минусы: отсутствие собственно нормального приложения для компьютера, в браузере доступен практически только из календаря, ну и функционал бедноват и не очень удобен для активной работы.
