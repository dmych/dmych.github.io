# Коллекции Bullet Journal

Bullet Journal состоит из модульных блоков — коллекций. Каждый модуль организует и коллекционирует определенную информацию.

Список покупок — коллекция, план ремнота — тоже, список книг для чтения на лето — тоже коллекция. Ну и список дел и событий на день — это тоже коллекция. Коллекции могут включать в себя другие коллекции (план на месяц включает в себя планы всех дней, его составляющих, план ремонта может включать в себя эскизы, списки материалов, которые надо купить, детальные планы отдельных этапов и т.п.).

В электронном виде для каждой коллекции логично использовать одну заметку/страницу/файл (в зависимости от программы, которую вы используете).
