# Vim

Лучший текстовый редактор, особенно для мобильных устройств (я имею в виду порт DroidVim).

DroidVim давно исчез из Google Play и F-Droid, можно загрузить только из репозитория на GitHub, но там версия без питона и git, которые были платными дополнительными опциями в версии для Google Play.

VimWiki — хорошая, удобная, настраиваемая система ведения заметок
