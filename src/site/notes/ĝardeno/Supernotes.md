# Supernotes
Приложение для ведения заметок по методу Zettelkasten. Я узнал о нем из блога «Федоров пишет». Заметки могут быть структурированы иерархически, причем у заметки может быть несколько родителей (т.е. она может находиться одновременно в нескольких темах, например), ближайшие заметки все перед глазами, структура — слева в панели (причем вы сами можете определять какие заметки должны там отображаться).

Для форматирования используется Markdown, поддерживаются ссылки между заметками, вставка картинок, простые таблицы, блоки кода и т.п. Есть поиск и фильтрация заметок. Выглядит стильно, использовать удобно (в том числе за счет хоткеев для большинства операций).

Есть мягкое ограничение на размер заметки — 1200 символов. Оно именно, что мягкое — никто не мешает делать заметки длиннее 1200 символов, вам просто внизу показывается, сколько символов осталось.

Приложение активно развивается. Есть версии для мобильных платформ и для компьютеров, а так же веб-приложение.
К основным недостаткам можно отнести отсутсвие офлайн-режима и стоимость.

Без интернета работать с заметками не получится вообще. Офлайн есть в планах на будущее, но не в ближайшее время.

Формально есть бесплатный тариф, но он ограничен только 40 заметками, т.е. по сути это просто триальная версия. Полноценно можно работать только на платном аккаунте, который стоит 6 фунтов в месяц при покупки годовой подписки.
