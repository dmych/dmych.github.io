# Hypernotes

Похоже на аутлайнер в стиле [[22.30 Заметки о заметокоделии/Dynalist\|Dynalist]]. Или, скоре, на [[22.30 Заметки о заметокоделии/Acreom\|Acreom]].

Заметки распределены по блокнотам. На бесплатном аккаунте можно только два блокнота.

Блокнот состоит из страниц. Каждая страница — список блоков (буллетов). Можно добавлять изображения и файлы. Есть таблицы, но, похоже, весьма простые. Есть ссылки на другие страницы и на отдельные блокию

Похоже, ссылок между блокнотами нет!
