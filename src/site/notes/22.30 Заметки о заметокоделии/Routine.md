# Routine

Интересное минималистичное приложение для дел и задач, но нельзя загрузить картинки (только URL), на бесплатном аккаунте не доступны виды календаря (на месяц, в частности), только таймлайн дня.

Есть страницы (заметки), на них можно добавлять текст и задачи, задачи потом появляются во входящих, их можно назначить на дату.
