# Daily logs в Obsidian

![Pasted image 20260226181414.png|400](/img/user/resursoj/Pasted%20image%2020260226181414.png)
Ежедневные заметки у меня создаются с именами в формате `YYYY-MM-DD`, все заметки месяца лежат в папке месяца (`YYYY-MM`), и все это собрано в папки по годам (`YYYY`). Чтобы плагин Daily notes в Обсидиане корректно работал с ежедневными заметками, нужно установить параметр Date format в `YYYY/YYYY-MM/YYYY-MM-DD` в его настройках.
